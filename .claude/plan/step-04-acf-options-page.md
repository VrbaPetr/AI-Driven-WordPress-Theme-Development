# Step 04 — ACF Global Options Page

**Phase:** 1 — Foundation  
**Depends on:** Step 01  
**Required by:** Steps 05, 06, 29, 33

---

## Summary

Register an ACF Options page ("Site Settings") and generate the field group JSON file directly into `acf-json/`. The fields store global site data — company info, contact details, social links, footer text, and a global CTA — that is reused across multiple templates without duplication.

---

## User Stories

- **As a site editor**, I want a single admin page to update the company phone number so it changes everywhere on the site simultaneously.
- **As a developer**, I want all global site data in `get_field('field_name', 'option')` calls so templates have a single source of truth.
- **As a site editor**, I want to control social media links from one place so adding or removing a network updates all components that display them.

---

## Business Value

Without an options page, global details (phone number, address, social links) get scattered across widgets, the Customizer, and hardcoded template strings. The options page is the canonical source, reducing content inconsistency across client handoffs.

---

## Acceptance Criteria

- [ ] ACF Options page registered programmatically, appears in WP admin under Settings → Site Settings.
- [ ] All field groups for the options page are generated directly into `acf-json/`.
- [ ] Fields are accessible in templates via `get_field('field_key', 'option')`.
- [ ] The following field groups and fields are present (see Field Specification below).
- [ ] Helper function `aidriven_get_social_links()` in `inc/functions-helpers.php` returns the social links array from options, ready for iteration in templates.

## Field Specification

All fields live on the **Site Settings** options page.

### Group: Company Info

| Field label | Field type | Notes |
|---|---|---|
| Company Name | Text | Used in footer, OG tags |
| Tagline | Text | Short strapline |
| Logo (Light) | Image | Used on dark backgrounds |
| Logo (Dark) | Image | Used on light backgrounds |

### Group: Contact Details

| Field label | Field type | Notes |
|---|---|---|
| Address | Textarea | Multi-line postal address |
| Phone Number | Text | Click-to-call; store with `tel:` strippable |
| Email Address | Email | Contact email displayed in footer/contact page |
| Google Maps URL | URL | Embed or directions link for contact page |

### Group: Social Links

Repeater field named **Social Links** with sub-fields:

| Sub-field label | Field type | Notes |
|---|---|---|
| Platform | Select | LinkedIn, GitHub, Twitter/X, Facebook, Instagram, YouTube, Other |
| URL | URL | Full profile URL |
| Label | Text | Screen-reader label (e.g. "Follow us on LinkedIn") |

### Group: Footer

| Field label | Field type | Notes |
|---|---|---|
| Footer Description | Textarea | Short company description shown in footer col 1 |
| Copyright Text | Text | e.g. "© 2024 Company Name. All rights reserved." |

### Group: Global CTA

| Field label | Field type | Notes |
|---|---|---|
| CTA Heading | Text | Banner headline |
| CTA Subtext | Text | Supporting sentence |
| Primary Button Label | Text | |
| Primary Button URL | URL | |
| Secondary Button Label | Text | Optional |
| Secondary Button URL | URL | Optional |

## Usage Pattern in Templates

```php
$company_name = get_field( 'company_name', 'option' );
$social_links = aidriven_get_social_links(); // returns array from options repeater
```
